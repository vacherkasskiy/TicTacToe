# Крестики-нолики

Простая консольная реализация классической игры "Крестики-нолики" на Python.

## Описание

Игра предназначена для двух игроков, которые по очереди делают ходы на поле 3×3. Первый игрок ставит крестики (X), второй — нолики (O). Цель игры — первым выстроить три своих символа в ряд по горизонтали, вертикали или диагонали.

## Функциональность

- **Консольный интерфейс** с отображением игрового поля
- **Валидация ввода** координат
- **Проверка корректности хода** (занята ли ячейка)
- **Автоматическое определение победителя** или ничьей
- **Поочередные ходы** двух игроков

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скопируйте код программы в файл (например, `main.py`)
3. Запустите программу командой:

```bash
python main.py
```

## Как играть

1. После запуска программы отобразится пустое игровое поле
2. Игроки ходят по очереди
3. Для выполнения хода необходимо ввести координаты ячейки в формате:
   ```
   X Y
   ```
   Где X и Y — числа от 0 до 2:
   - **X** — номер строки (сверху вниз)
   - **Y** — номер столбца (слева направо)

   Пример: `0 0` — левый верхний угол, `1 2` — средняя строка, правый столбец

4. Игра продолжается до:
   - Победы одного из игроков
   - Заполнения всех ячеек поля (ничья)

## Особенности реализации

- Поле отображается с нумерацией строк и столбцов для удобства
- Система автоматически определяет, чей сейчас ход
- Проверка на корректность введенных данных
- Защита от хода в уже занятую ячейку

## Структура кода

Основные функции:
- `start_game()` — главная функция игры
- `make_turn()` — обработка хода игрока
- `print_grid()` — отображение игрового поля
- `check_win()` — проверка условий победы
- `get_coords()` — получение и валидация координат от игрока

## Ограничения

- Игра рассчитана только на двух игроков
- Отсутствует компьютерный оппонент
- Консольный интерфейс без графики
